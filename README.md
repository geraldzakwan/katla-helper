# Katla Helper

A web app (currently just the backend/API) that can help you solve Katla game
(https://katla.vercel.app/) by giving 5 suggestions for each step.

# How to Run

- Install all requirements

  `pip install -r requirements.txt`

- Run the web app

  `python3 app.py`
